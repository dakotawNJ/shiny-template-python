## Base template for NJ DEP Python Shiny Apps
### Download and extract the repository files:
- Under the **Code** tab, click the **Code** dropdown
- Click **Download ZIP**
- Extract the the **.zip** file into your project directory

### Virtual environment setup:
- Create your virtual environment: `py -m venv .venv`
- Active the environment: `.venv\Scripts\activate.bat`
- Install **shiny**: `py -m pip install -r requirements.txt`
